0. 현재 notebook python 버전 확인
윈도우 터미널에서 
$python --version

1. 가상환경 설치
Rpi python version >> python 3.9.2 32-bit

2. python 설치
https://www.python.org/ 접속 
>Downloads > 3.9.2 > 아래로 내려서 Windows installer (32-bit) 설치 > customized version
>설치 경로 기록

3. 환경 변수 설정
제어판 > 시스템> 고급 시스템 설정 > 환경변수 > path > 경로 추가 > 설치 경로 저장
*만약 환경 변수 설정 후 파이썬 버전 체크시 기존 버전 정보가 아니라 3.9.2로 변경되었다면,
기존 버전 정보의 python을 찾아서 환경변수로 재등록하고, 파이썬 3.9.2 보다 위로 이동시키기(우선순위)

4. 적절한 위치에 다음과 같이 실행 (powershell 기준)
가상환경 설치 	$py -3.9 -m venv [가상환경이름]
가상환경 	실행 	$.\[가상환경이름]\Scripts\activate
실행시 (가상환경이름) 경로> cmdline 형태로 실행되면 정상 작동
버전 체크시 정상적으로 변경된 것을 확인 가능
가상환경 종료	$deactivate

5. kivy 설치
가상환경 실행(activate)
$python -m pip install --upgrade pip setuptools
$python -m pip install "kivy[base]" kivy_examples
